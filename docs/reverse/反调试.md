### 断点检测
检测指令是否为0xcc
```c
void foo() {
    printf("Hello\n");
}
int main() {
    if ((*(volatile unsigned *)((unsigned)foo) & 0xff) == 0xcc) {
        printf("BREAKPOINT\n");
        exit(1);
    }
    foo();
}
```

#!/usr/bin/perl
while(<>)
{

}
