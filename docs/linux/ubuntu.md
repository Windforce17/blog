## 如何重启 php

启动 php-fpm:

/usr/local/php/sbin/php-fpm
INT, TERM 立刻终止
QUIT 平滑终止
USR1 重新打开日志文件
USR2 平滑重载所有 worker 进程并重新载入配置和二进制模块
先查看 php-fpm 的 master 进程号
kill -USR2 \<master pid\>
